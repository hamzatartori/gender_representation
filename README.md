This repository contains the bibliometric metadata used in the analysis of research on gender representation in media and communication studies published between 2015 and 2024.

## Contents
- `metadata.bib`  
  A single BibTeX file containing the publication records used for the bibliometric and content analysis reported in the associated article.

## Description
The dataset includes 2,225 bibliographic records retrieved from major academic databases and was used to examine publication patterns, leading sources, collaboration networks, and thematic trends in gender representation research across traditional and digital media contexts.

The repository is intended to support transparency, reproducibility, and reuse of bibliometric data for scholarly purposes.

## Notes
- The file contains bibliographic metadata only (e.g., titles, authors, abstracts, keywords, sources, citations).
- No full-text articles or copyrighted content are included.
- The data are provided as used in the analysis and have not been modified beyond standard cleaning and deduplication procedures.
